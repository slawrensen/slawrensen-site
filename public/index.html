HWiNFO Sensors for Stream Deck

A Stream Deck plugin that puts live HWiNFO hardware sensors on your keys and dials, updated every second.

Windows / Elgato Stream Deck

Your hardware, at a glance

Put CPU and GPU temperature, clock speeds, fan RPM, usage and power on your keys and dials.

  • KeysAny Stream Deck. One sensor per key, with a live sparkline.
  • DialsStream Deck + and Stream Deck + XL: rotate, push and touch, rotation sets with named groups, and a Control key that drives dials from anywhere. Hardware compatibility
  • DisplaySeven themes, including a high-contrast light theme for bright rooms and low vision. Amber warn and red critical alerts keep the same palette on every theme, so an alert stays unmistakable with any colour vision.
  • StatusWhen HWiNFO stops publishing, keys say so: every failure state renders a screen that names the problem and its fix, instead of a stale number pretending to be live.
  1. Install the plugin from the Elgato Marketplace, or double-click the .streamDeckPlugin from GitHub Releases.
  2. Run HWiNFO (free or Pro) and tick Shared Memory Support in its Settings. HWiNFO and the Stream Deck app must run at the same privilege level — both elevated or both normal.
  3. Drag Sensor Reading onto a key and pick a sensor from the searchable list. The key goes live immediately.

Free HWiNFO switches Shared Memory off after 12 hours. If you have Gadget reporting enabled, the plugin falls back to it automatically and upgrades itself back the moment Shared Memory returns. HWiNFO is a separate free program by REALiX; this plugin does not bundle it.

About

Built by one person, on purpose.

I am slawrensen, an independent developer. I build small, honest tools for people who care how their machines run.

Every product here is made to be fast, quiet, and trustworthy. If it reads your hardware, it does so locally and tells you exactly what it does.

The work is open source, so you never have to take my word for any of it. The code is the documentation of intent.

HWiNFO Sensors is a ground-up TypeScript rewrite on Elgato's official SDK, inspired by the archived Go-based plugin by @shayne — no code shared, credits in NOTICE.md.

Questions, bugs, or ideas go through GitHub Issues, where they are answered in the open.
